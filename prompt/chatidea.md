## 目标
给企业内部使用的LLM API Gateway平台新增一个AI Chat界面。

## 主要功能：
可以让企业内部员工使用自己在LLM API Gateway平台申请的key直接在UI中与AI进行沟通对话

## 模块规划：

- 核心模块
  - AI Chat
    - 是否有UI：有UI
    - 用户角色：系统管理员、团队管理者、普通用户
      - 主要功能：
        - 1. 可以从已申请的key列表选择一个key
        - 2. 可以从平台提供的模型列表选择一个模型
        - 3. 可以与AI进行对话，输入提示词并接收模型返回的结果
        - 4. 可以查看对话历史，支持对话的保存和加载
        - 5. 可以与AI进行多轮对话，支持上下文理解和对话状态保存
        - 6. 可以与AI进行文件上传，支持文件内容理解和对话状态保存
