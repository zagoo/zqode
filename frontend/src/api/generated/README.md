# Generated API client

This directory is intended to be populated by `@hey-api/openapi-ts` against the
FastAPI `/openapi.json` endpoint.

```bash
npm run openapi:gen
```

Until that command is run, the hand-written stub at `client.ts` and the helper
in `src/api/http.ts` provide a working baseline so the SPA can bootstrap before
the backend is reachable.

Do NOT manually edit files in this directory — regenerate instead.
